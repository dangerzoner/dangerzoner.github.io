==> bookmarklets

        bookmarklets are bookmarks that contain special javascript code that will execute script instead of redirecting onto another page

    there are varieties of bookmarklets, some allow you to do math calculations in a quick way, some allow you to play minigame, some add cool effect to the website

    bookmarklets are usually used as a quick and portable tool since you usually only have to click once to run the code and it's very fast as well

    not sure why but i see a sudden decline in bookmarklets these days despite they are helpful up to this day

    *maybe the fact that some sites block the bookmarklets to directly run scripts may have something to do with it

    (you can bypass it by copying the bookmarklet code through copying the link and paste it into the inspect element's console)

 

warning : only use bookmarklets you trust like sites on google (not the ad ones)

for whatever reasons, do not run bookmarklets made by random people since they are basically javascript code that can do a lot of things

it's up to you to use these or not, please report to me if any javascripters here saw anything weird in these bookmarklets

--<<==     jsgif     ==>>--

made by shachaf@gmail.com

 

[[jsgif website here]]

================

description : this is a special scripted bookmark that allows you to choose a gif on the website you are own and pause, play, go through the gif frame by frame as well as saving the frame into file (usually under .png)

 

[pros]

[cons]

easy to use, just click on a gif and use it like a media player

does not work properly on cumulative gifs (lots of black spots)

best used for x-ray zap gifs to properly see the skeleton

sites with hotlink protection like furaffinity requires the gif to be opened on a new window

gif can be played in reverse and viewed frame by frame

bigger gifs can take a long time to load, especially high res

it's a bookmark so no download needed, just click and use

you might need to zoom out on some sites to see control

made by http://www.javascripter.net/

 

[[bookmarklet calculator website here]]

================

description : this is a small and portable calculator bookmarklet that let you do math equations in a very quick way, all you really have to do is click on the bookmarklet, type an equation in like "5*2+3" or "256/2" and you will get your answer after you press enter

 

[pros]

[cons]

easy to use and fast without any loading

== instruction ==

(too lazy to create any graphics work at the moment)

3. drag the hyperlink into the bookmark bar (below your url bar)

5. click on it to start doing a calculation and that's very much all

--<<==    bookmarklet pong     ==>>--

made by dragonmaster/dragon731012 ?

 

[[bookmarklet pong on github]]

================

description : a bookmarklet version of pong, it's quite janky but can be fun if you are using it in a boring place like during class maybe, the game starts off slow but after every win / lose, the bot will be faster and the game will be harder as well (you can restart difficulty through reloading the page), also if the "ball" touches the side part of the paddle, it will either shoot out way faster than before or it will move behind the paddle, making you lose the game (if you experience stuttering "ball", enable hardware acceleration on your browser)

 

[pros]

[cons]

fun little minigame during boring time

can be quite boring after a bit

can be fun during harder difficulty

first match bot is absolutely boring

you can speed the ball up by letting the ball touch the side

doing so have a chance of backfiring, making you lose the game

 

== instruction ==

 

(unlike the other bookmarklets, this one is not a bookmark yet so you will have to set it up yourself)

1. go to the github page here

2. on the right side, there should be a "copy raw file" button depicting as 2 squares

3. find bookmark manager (for chrome it's in 3 dots -> bookmarks -> bookmark manager)

4. create a new bookmark, name it to however you want

5. paste the code you copied earlier into url, make sure there's javascript: before the code itself

(check through select all using ctrl+a and press left arrow)

6. save the bookmark and drag it onto the bookmark bar (below the url bar)

7. go on google, click on the bookmark and have fun
